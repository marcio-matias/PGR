# PGR - Programa de Gerenciamento de Riscos

Projeto web para elaboração e organização de um PGR, com foco em cadastro de empresas, texto base, inventário de riscos, EPIs, medidas de controle e plano de ação.

## Objetivo

Criar uma ferramenta simples em HTML, CSS e JavaScript para estruturar informações de um Programa de Gerenciamento de Riscos.

## Módulos previstos

- Empresa contratante
- Empresa contratada
- Dados do PGR
- Histórico de revisões
- Texto base do PGR
- Inventário de riscos
- EPIs vinculados
- Medidas de controle
- Plano de ação
- Impressão / salvar como PDF
